# Geo Mini — геометрический редактор

Веб-приложение для построения геометрических фигур на холсте: точки, отрезки, окружности, многоугольники и др. Состоит из backend (FastAPI) и frontend (React + Vite).

## Требования

- Python 3.10+
- Node.js 18+

## Запуск

### 1. Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

API будет доступен по адресу: http://127.0.0.1:8000

### 2. Frontend

В отдельном терминале:

```bash
cd frontend
npm install
npm run dev
```

Приложение откроется по адресу: http://localhost:5173

> Backend и frontend должны работать одновременно.

## Использование

1. Зарегистрируйтесь или войдите в аккаунт.
2. Создайте холст на главной странице.
3. Стройте фигуры через панель «Построение» (мышью) или вводите команды на вкладке «Команды».

## Команды

Формат: `Имя = Операция(аргументы)` или `Операция(аргументы)` — имя назначается автоматически.

У назначенного автоматически имени точки обозначаются заглавными буквами (`A`, `B`), линии — строчными (`a`, `b`), фигуры конкастинацией вершин (`ABC` `ABCD`).

### Служебные

| Команда | Описание |
|---------|----------|
| `Clear()` | Очистить холст |
| `Delete(A)` | Удалить объект и все зависимые от него |

### Точка

```
A = Point(0, 0)
Point(3, 4)
```

### Отрезок

```
a = Segment(A, B)
a = Segment(0, 0, 3, 4)
a = Segment(A, 3, 4)
```

### Луч и прямая

```
b = Ray(A, B)
b = Ray(0, 0, 3, 4)
b = Ray(a)          — по отрезку a

c = Line(A, B)
c = Line(0, 0, 3, 4)
c = Line(a)         — по отрезку a
```

### Окружность

```
d = Circle(A, B)    — центр A, точка на окружности B
d = Circle(A, 5)    — центр A, радиус 5
d = Circle(0, 0, 5) — центр (0, 0), радиус 5
d = Circle(0, 0, 3, 4) — центр (0, 0), точка (3, 4)
```

### Квадрат

```
e = Square(A, B)              — сторона AB
e = Square(0, 0, 2, 0)        — по координатам двух вершин
e = Square(A, B, C, D)        — по четырём вершинам
```

### Треугольник

```
f = Triangle(A, B, C)
f = Triangle(0, 0, 3, 0, 1, 2)
```

### Ромб и параллелограмм

```
g = Rhombus(A, B, C)          — сторона AB и точка направления C
g = Rhombus(A, B, C, D)       — по четырём вершинам

h = Parallelogram(A, B, D)    — сторона AB и вершина D
h = Parallelogram(A, B, C, D) — по четырём вершинам
```

### Полигон

```
i = Polygon(A, B, C, D)
i = Polygon(0, 0, 2, 0, 3, 1, 1, 2)   — минимум 3 точки
```
